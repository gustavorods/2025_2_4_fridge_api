# Fridge Project

**Tagline**: Refrigerator design project.

## Description

A project that stores, deletes, and displays cataloged items inside a refrigerator.

## Installation

To set up the project, clone the repository:

**Example:**

```bash
https://github.com/gustavorods/2025_2_4_fridge_api.git
```

## Usage

- After cloning the project, navigate into the project folder and run `FridgeApplication.java`.

- After the server starts up, it will be available at the URL `localhost:8080`.

- You can also access the bank via the route `/h2-console`.

- You can find/change the credentials in the file `application.yml`.
